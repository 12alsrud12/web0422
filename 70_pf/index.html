진에어(2024.05.28 ~ 2024.07.02)

수업시간 중에서만 진행하는 "팀 프로젝트" 진에어 사이트 리뉴얼 프로젝트 입니다. 제가 맡은 부분은 "예약/결제안내"와 "회원가입" 파트부분이고, 예약/결제안내는 이때는 아직 JS를 배우기 전이라서 " html과 css만"을 사용하여 탭 형식으로 제작하여 예약 부분과 결제 안내 부분을 한 파일 내에 작성해서 버튼을 누르면 전환하는 식으로 하였습니다.

main page 기준에서 예약/결제안내는 헤더 부분의 "나의 예약" click 시 확인이 가능하고 회원가입은 헤더의 오른쪽 위 맨 끝 사람 모습을 누르면 로그인 페이지가 나오는대 거기서 회원가입을 클릭 시 회원가입 페이지가 나타납니다. 그리고 자바스크립트 막 배울때 쯤 조별과제가 끝나서 JS 코드 alert를 부분적으로 추가해보았습니다.
